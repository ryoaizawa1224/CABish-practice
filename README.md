# CABish Practice — CAB形式Webテスト練習ツール

CAB（Computer Aptitude Battery）形式の就活Webテストを、本番に近い環境で練習できるデスクトップアプリです。

---

## ダウンロード

👉 **[最新版をダウンロードする（Releases）](https://github.com/ryoaizawa1224/CABish-practice/releases/latest)**

| OS | ファイル |
|----|---------|
| Mac（M1 / M2 / M3） | `CAB.-x.x.x-arm64.dmg` |
| Mac（Intel） | `CAB.-x.x.x.dmg` |
| Windows | `CAB.Setup.x.x.x.exe` |

### Macでのインストール手順
1. `.dmg` ファイルを開く
2. アプリをApplicationsフォルダにドラッグ
3. 初回起動時に「開発元が未確認」と出たら → **システム設定 → プライバシーとセキュリティ → 「このまま開く」**

### Windowsでのインストール手順
1. `.exe` ファイルを実行
2. 「WindowsによってPCが保護されました」と出たら → **「詳細情報」→「実行」**

## 収録テスト（本番CABの形式に準拠）

| テスト | 問題数 | 制限時間 | 内容 |
|--------|--------|----------|------|
| **四則逆算** | 50問 | 9分 | `□ + 3 = 7` のような穴埋め計算（5択） |
| **法則性** | 30問 | 12分 | 5つの図形の規則を見抜いて6番目を予測（5択） |
| **命令表** | 36問 | 15分 | 初期図形列に命令を順に適用、最終形を選ぶ（5択） |
| **暗号** | 30問 | 16分 | 変換ヒント例から暗号の意味を推論し、新しい入力の結果を選ぶ（5択） |

### 特長

- ✅ **完全無料**・登録不要
- ✅ **問題は毎回ランダム生成**（無限に練習可能）
- ✅ Mac / Windows のデスクトップアプリ
- ✅ 本番CABと同じ**制限時間・問題数・5択選択式**
- ✅ 途中で**中断してテスト選択画面に戻れる**
- ✅ オフライン動作（API・ログイン不要）

### 各テストの問題形式

#### 四則逆算
```
□ + 47 = 92    →  ア:42  イ:43  ウ:44  エ:45  オ:46
```

#### 法則性
5つの図形が並んでいて、6番目に来る図形を5択から選ぶ。
回転・反転・形の交替・塗りつぶし反転などのパターン。

#### 命令表
初期図形列（例: ▲ ◆ ★）に対して、命令（左右反転・上下反転・塗り反転・先頭/末尾削除・回転・入替・取消）を順に適用し、最終形を5択から選ぶ。

#### 暗号
ヒント例（▲ → [★] → ▼ など）が複数提示され、★の意味（変換ルール）を推論。その上で、別の図形に★を適用した結果を5択から選ぶ。

## 開発者向け：ローカルで動かす

```bash
# リポジトリをクローン
git clone https://github.com/ryoaizawa1224/CABish-practice.git
cd CABish-practice

# 依存パッケージをインストール
npm install

# デスクトップアプリとして起動（ビルド込み）
npm run electron
```

### 新しいリリースを出す方法

```bash
git tag v1.x.x
git push origin v1.x.x
```

タグをpushするとGitHub Actionsが自動でMac・Windows向けのビルドを作成し、Releasesに公開します。
